# Git Version Control Setup Plan

## 1. Configure `.gitignore`
Update `.gitignore` to exclude unnecessary files specifically for WeChat Mini Programs and macOS:
- **System Files**: `.DS_Store`, `._*` (macOS temporary files)
- **Dependencies**: `node_modules/`, `miniprogram_npm/` (generated by tools)
- **Local Config**: `project.private.config.json` (contains user-specific settings)
- **Logs**: `npm-debug.log`, `yarn-error.log`

## 2. Initialize & Configure Git
1. Initialize git repository (if not fully set up).
2. Add the remote repository URL: `https://github.com/1924605670/force_time.git`
3. Create the `main` branch.

## 3. Commit & Push
1. Stage all files (respecting the new `.gitignore`).
2. Commit with message "Initial commit: WeChat Mini Program structure with AI features".
3. Push to the remote repository.
